# 🏫 AI & Data School (ai-data-school)

> **AI를 도구로 활용하여 스스로 호기심을 탐색하고, 교과 연계 수행평가 및 숙제 산출물을 도출하는 통합 교육과정 강사 전용 저장소입니다.**

본 교육과정은 복잡한 텍스트 기반 코딩 입력에서 발생하는 인지적 부담을 줄이기 위하여, 학생이 지시어를 설계하고 인공지능이 코드를 생성하는 **바이브 코딩(Vibe Coding)** 방식을 핵심 교수 방법으로 적용합니다. 단순한 데이터 분석가 양성이 아닌, **AI를 활용한 논리적 완결성을 갖춘 학교 산출물 제작**을 목적으로 합니다.

---

## 🚀 학년별 커리큘럼 요약 (Curriculum)

* **초등 3~4학년 [데이터베이스 기초]** : SQLite 및 DBGate 활용, 데이터 수집·조회 및 기초 시각화 보고서 작성
* **초등 5~6학년 [머신러닝 기초]** : Scikit-learn 기반 텍스트 마이닝, 회귀 및 분류 모델 활용 데이터 예측, MCP 연계 자동화 탐구 보고서 완결
* **중등 ~ 고등 [DB + ML 융합 고도화]** : 주도적 데이터 검색/수집부터 구조화(DB), 머신러닝 예측(ML), MCP 연계 자동화 탐구 보고서 완결

# 텍스트 마이닝을 활용한 소설 『운수 좋은 날』 분석

이 프로젝트는 파이썬(Python)과 자연어 처리 기술을 활용하여 소설의 텍스트 데이터를 분석하고 시각화하는 과정을 담은 융합 탐구 리포지토리입니다. 국어 교과 수행평가 및 데이터 분석 실습을 목적으로 작성되었습니다.

## 📌 주요 내용

- **자연어 처리(NLP)**: `KoNLPy`를 이용하여 소설 텍스트(`운수좋은날.txt`)에서 명사만 정확하게 추출합니다.
- **데이터 정제(불용어 처리)**: 분석에 방해가 되는 인물 이름(진수, 혜진 등)이나 의미 없는 단어(불용어)를 필터링하여, 작품의 주제를 관통하는 순수 핵심 키워드만을 도출합니다.
- **데이터 시각화**: 빈도수 상위 20개 단어를 **막대그래프**와 **워드 클라우드**로 시각화하여 작품의 전반적인 분위기를 직관적으로 파악합니다.
- **결과 저장 및 보고서 작성**: 분석된 단어 빈도수 결과를 엑셀 파일로 저장하고, 추출된 핵심 단어(설렁탕, 이혼, 운수 등)를 바탕으로 소설의 상징적 의미와 비극적 아이러니를 분석한 최종 보고서를 Word 문서로 작성합니다.

## 📂 파일 구성

- `main.ipynb`: 형태소 분석, 불용어 제거, 데이터 시각화 및 엑셀 저장 코드가 모두 포함된 메인 주피터 노트북입니다.
- `운수좋은날.txt`: 텍스트 마이닝 분석의 원본 대상이 되는 소설 텍스트 파일입니다.
- `출처.txt`: 텍스트의 원본 출처(공유마당) 정보를 담고 있습니다.
- `막대그래프.png`: 불용어 제거 후 도출된 상위 20개 단어의 빈도수를 나타낸 막대그래프 시각화 자료입니다.
- `워드클라우드.png`: 추출된 주요 단어들의 출현 빈도를 바탕으로 생성한 워드 클라우드 이미지입니다.
- `운수좋은날_단어빈도수.xlsx`: 상위 20개 단어와 그 빈도수를 표 형태로 정리한 데이터 파일입니다.
- `운수좋은날_텍스트마이닝_보고서.docx`: 추출된 핵심 단어들을 바탕으로 작품의 비극성을 객관적으로 분석한 국어 수행평가 제출용 최종 보고서입니다.

## 🚀 사용된 기술 스택

- **언어**: Python 3
- **라이브러리**: 
  - `KoNLPy` (한국어 형태소 분석)
  - `collections.Counter` (단어 빈도수 집계)
  - `matplotlib` & `wordcloud` (데이터 시각화)
  - `pandas` & `openpyxl` (데이터프레임 생성 및 엑셀 저장)

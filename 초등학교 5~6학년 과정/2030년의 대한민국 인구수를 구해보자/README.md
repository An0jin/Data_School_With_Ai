# 📊 2030년 대한민국 인구수 예측 및 사회 수행평가 자동화 프로젝트

본 프로젝트는 **코딩 교육 및 AI 활용 시연**을 목적으로 제작되었습니다. 학생들에게 파이썬(Python) 기반의 데이터 분석과 인공지능(AI) 어시스턴트를 활용하여, 학교 수행평가(사회 과목 등) 자료를 얼마나 빠르고 수준 높게 완성할 수 있는지 보여주기 위한 실습 예제입니다.

## 🎯 프로젝트 개요
과거부터 현재까지의 통계청 남녀 인구수 데이터를 바탕으로, **머신러닝(다항 회귀, Polynomial Regression)** 알고리즘을 적용하여 2030년의 대한민국 남녀 인구수를 예측합니다. 더불어 도출된 데이터를 바탕으로 저출산, 고령화, 여초 사회 진입 등 사회적 시사점을 분석하여 완성된 워드(Word) 보고서를 자동으로 생성합니다.

## 📁 주요 파일 설명

* **`인구수.csv`**
  * 대한민국 연도별 남녀 인구수가 기록된 원본 데이터 파일입니다.
* **`main.ipynb`**
  * 주피터 노트북 파일로, 데이터 전처리, 모델 학습, 2030년 예측, 그리고 시각화(그래프 도출)에 이르는 전체 데이터 분석 코드가 담겨 있습니다.
  * 수행평가 제출용에 맞게 각 단계별 설명이 마크다운(Markdown)으로 꼼꼼하게 작성되어 있습니다.
* **`인구수_예측_그래프.png`**
  * 파이썬 코드를 통해 도출된 과거 인구 변화 추이 및 2030년 예측선이 포함된 고화질 시각화 그래프입니다.
* **`사회_수행평가_보고서.docx`**
  * AI(Word MCP)를 통해 자동 생성된 최종 수행평가 보고서입니다.
  * 탐구 목적, 분석 방법, 시각화 자료, 그리고 **'2015년 이후 여초 사회로 역전된 원인(고령화 및 남아선호사상 퇴조)'**에 대한 분석과 결론이 포함되어 있습니다.

## 🛠 사용된 기술 및 라이브러리
* **언어**: Python 3
* **데이터 분석 및 시각화**: `pandas`, `numpy`, `matplotlib`, `koreanize-matplotlib`
* **머신러닝**: `scikit-learn` (선형/다항 회귀 알고리즘 적용)
* **AI 연동**: Word Document MCP Server (보고서 자동 작성 기능)

## 💡 교육적 기대 효과 (수업 활용 포인트)
1. **실용적인 코딩 경험**: 단순한 문법 위주의 학습을 벗어나, 학교 과제(수행평가)라는 학생들의 현실적인 문제 해결에 코딩이 어떻게 쓰이는지 보여줍니다.
2. **AI와의 협업 능력(Prompt Engineering)**: 코드를 처음부터 끝까지 직접 작성하는 것을 넘어, 기획 의도를 AI에게 정확히 전달하고 결과물을 다듬어 내는 현대적인 개발 워크플로우를 체험할 수 있습니다.
3. **데이터 문해력(Data Literacy)**: 단순히 '인구가 줄어든다'는 사실을 넘어, 데이터 속에서 '여성 인구가 남성 인구를 추월했다'는 숨겨진 패턴을 발견하고 그 사회적 원인을 탐구하는 통찰력을 기릅니다.
